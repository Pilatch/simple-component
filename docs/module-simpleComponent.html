Module: simpleComponent

Define a stateless custom element with no behaviors. Give the element's template an ID attribute with the "-template" suffix.
Parameters:
Name Type Attributes Description
elementName String Name of the custom element
elementClass Object <optional>
 An object with lifecycle callbacks on it. Not actually an ES6 class for compatability purposes.
elementClassToExtend function <optional>
 An existing HTML element class to extend, e.g. HTMLButtonElement.

Returns:

Example
<template id="simple-bling-template">
  <style>
    simple-bling {
      color: gold;
      font-weight: bold;
    }
  </style>
</template>
<script>
  simpleComponent('simple-bling')
</script>
